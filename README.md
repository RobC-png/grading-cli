# Student Grading CLI (C)

Interactive command-line tool that calculates a final grade based on
multiple assessment components such as intermediate tests, final tests,
and exercises.

The program dynamically tracks achievable points and determines the
final grade based on defined percentage thresholds.

## Features
- Incremental input of assessment results
- Validation of point ranges per assessment type
- Dynamic calculation of achievable and valid points
- Final grade computation based on percentage thresholds

## Concepts Demonstrated
- Procedural program structure
- Input validation
- Floating-point arithmetic
- Control flow and state management
- Modular function design
